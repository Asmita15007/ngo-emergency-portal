# 🏥 NGO Emergency Portal (Local AI)

A privacy-focused emergency coordination portal that uses a **Local RAG (Retrieval-Augmented Generation)** architecture to match volunteer resources with emergency needs.

## 🚀 Key Features
*   **Local AI:** Powered by Llama 3.2 via Ollama (No data leaves your device).
*   **Role-Based Access:** Separate dashboards for Volunteers (Upload) and Staff (Search).
*   **Smart Retrieval:** Uses Vector Embeddings to find the most relevant aid records.

## 🛠️ Setup Instructions
1. **Install Ollama:** Download from ollama.com and run `ollama run llama3.2`.
2. **Clone Repo:** `git clone https://github.com/Asmita15007/ngo-emergency-portal.git`
3. **Install Dependencies:** `pip install -r requirements.txt`
4. **Run App:** `streamlit run auth.py`
